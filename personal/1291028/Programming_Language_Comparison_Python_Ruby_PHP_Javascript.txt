Python

Python은 1991년프로그래머인 귀도 반 로섬이 발표한 고급 프로그래밍 언어로, 플랫폼 독립적이며
인터프리터식, 객체지향적, 동적 타이핑(dynamically typed) 대화형 언어라고 합니다. 파이썬의 주요 특징은 동적타이핑과
객체의 멤버에 무제한 접근이 가능하며 모듈, 클래스, 객체와 같은 언어의 요소가 내부에서 접근할 수 있고, 리플렉션을
이용한 기술을 쓸 수 있습니다. 

Python의 장점 
첫번째 장점은 코드가 직관적이기 때문에 이해하기 쉬워 초보자도 배우기 쉽습니다. 두번째 장점은 다른 프로그래밍과의 연동이 
용이 하다는 점 입니다. 세번째 장점은 C언어에서는 몇 백줄이 되는 코딩도 백 줄 이하로 간단하게 코딩 할 수 있습니다. 
즉, 코딩이 간결하기 때문에 개발 시간이 단축됩니다. 네번째 장점은 메모리가 자동관리되어 유지보수가 쉽습니다. 
마지막으로 다섯번째 장점은 수많은 라이브러리를 제공하는 점 입니다.



Ruby

Ruby는 마츠모토 유키히로가 개발한 동적 객체 지향 스크립트 프로그래밍 언어입니다. 또한 루비는 순수 객체지향 언어라서
모든것이 객체라고 합니다.

Ruby의 장점
포스트 모던 프로그래밍의 언어로서 Ruby의 첫번째 장점은 프로그래밍 언어들의 장점들을 받아들여 만들어진 실용적인 프로그래밍
언어입니다. 두번째 장점은 Ruby의 꽃이라 할 수 있는 블록이라는 기능으로 스크립트를 변수처럼 저장해두고 원할 때 실행할 수 
있습니다. 세번째 장점은 변수 선언이 필요없어 모든 변수를 선언 없이 갑자기 사용할 수 있고 변수 명을 보는 것으로도 변수가 
로컬인지 글로벌인지 인스턴스인지 구별할 수 있습니다. 네번째 장점은 정형화 되어 있지 않아 문법이 자유롭다는 것입니다.
마지막으로 다섯번째 장점은 배우기 쉽다는 점입니다



PHP

PHP(PHP: personal Hypertext Preprocessor)는 원래 동적 웹페이지를 설정하기 위해 만들어졌습니다.

PHP의 장점
첫번째 장점은 빠른 생산성으로 PHP는 프로그래밍 속도를 높여주기에 좋아서 빠른 생산성을 가지고 있다. 두번째 장점은 함수들도
직관적인 것이 많아 편하게 프로그래밍을 할 수 있다. 세번째 장점은 PHP는 C 언어를 기반으로 만들어진 스크립트언어로 C 언어에
익숙한 사용자들은 보다 쉽게 배울수 있다는 점이다. 네번째 장점은 유닉스 계열 및 윈도우 환경등 모든 운영 체제에서 구현이 
가능하다는 점입니다. 마지막으로 다섯번째 장점은 데이터베이트 연동이 편리한 점입니다. MySQL,mSQL,Oracle,Sybase 및 
윈도우 ODBC 등 여러 데이터베이스를 편리하게 연동할 수 있다는 수 있습니다.



Javascript

Javascript는 객체 기반의 스크립트 프로그래밍 언어입니다. 

Javascript의 장점
첫번째 장점은 Javascript는 html 소스 코드 안에 그대로 끼워 넣어 따로 컴파일 하지 않고 그대로 브라우저로 실행할 수 있습니다.
두번째 장점은 유닉스나 윈도우즈, 리눅스 등 운영체제에 제한없이 사용이 가능하다는 점 입니다. 세번째 장점은 Javascript는 
메모장이나 기타 다른 프로그램으로 바로 사용이 가능하다. 즉, 별도의 프로그램 없이 사용할 수 있습니다. 네번째 장점은 
애니메이션 효과를 줄 수 있다는 점입니다. 이벤트를 이용해 마우스로 그림을 움직일 수도 있고 글에 효과를 주는등 다양한 
효과를 줄 수 있습니다. 마지막으로 다섯번째 장점은 웹브라우저에서 실행이 바로 되기 때문에 처리속도가 빠르다는 점입니다.
