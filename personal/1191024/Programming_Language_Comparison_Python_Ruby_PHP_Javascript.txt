파이썬이라는 언어는 프로그래밍을 배우면서 처음 들어 본 언어이다. 파이썬에 대해 검색 해 본 결과 파이썬은 내가 알고있는 프로그래밍 언어 중에 문법이 그나마 쉬운 듯 하다. 처음 배우는 사람도 이해하기 쉬울 정도로 간단하고 핵심적이게 해놔서 처음 입문 할 때 쉽게 배울 수 있을거같다. 두번째로는 C언어 처럼 윈도우에서만 되는 언어와는 달리 파이썬은 범용성이 높다. 윈도우,맥,유닉스,노키아 모두 에서 사용이 가능하다고 한다. 그리고 마지막으로 가장 큰 장점은 라이브러리가 제공된다고 한다. HTTP,DB접속,GUI인터페이스 등의 도구를 포함하며, 라이브러리가 없을 때에는 자기가 만들어서 붙여서 사용이 가능해서 대부분의 작업에 사용 할 수 있다고한다. 그리고 우리처럼 c언어를 먼저 배운 사람이라면 더욱 더 쉽게 파이썬을 사용할 수 있다고한다.그리고 뒤에나오는 php 는 웹에 특화되어있어, 안되는게 많은데 파이썬은 무엇이 든 간단하게 모든 걸 할 수있다고한다.

Ruby 어떤 한 개발자가 루비언어로 개발을 해보면서 느낀 장점이라고한다. 1.MVC모델이 완벽히 구현 된다고 하는데 MVC모델을 몰라 찾아본 결과 MVC 모델은 Model View Control 의 약자로 웹 앱을 만들 때에 역할마다 구분하여 프로그램 하는 뜻이라고 한다. 루비언어는 어떠한 추가작업 없이 도, 폴더 안에 있는 파일의 이름만 맞추면 거의 대부분 MVC 모델이 만들어진다고 한다. 그래서 구조화 작업이 최 소화 된다는 장점이있고, 두번째로 2.외부 라이브러리를 추가 하는 것도 간단하다고 한다. 특정한 파일에 라이브러 리명,버전만 명시 하고 , 설치를 하면 끝이다. 자바에 비해서는 라이브러리가 적다고하지만 웹 개발로써는 충분하 다고 본다.그리고 3.기타 귀찮은 작업들은 최소화 해주며, 4.루비언어도 자바나 c에 비해서 쉽고, 빠르게 개발이 가능하다고한다.가장 큰 장점은 4. 속도가 엄청 빠르다는 것이다.

PHP 이언어도 위에 언어랑 마찬가지로 1.배우기 쉽다고한다. 아마도 자바나 C에 비해서 쉽다는거같다.그렇다고 언어 자체가 쉬운건 아니고, 접하기에만 쉽다고한다. 그리고 2.PHP도 가볍기 때문에 속도는 빠르다고하고, 3.무료로 제공되는 언어지만 강력하다고한다.그리고 지금 배우고있는 data base에서 사용하는 4.MySQL이나 자바스크 립트와 상호연동이 아주 많이 뛰어나다고한다. 그리고 대부분의 5.데이터베이스, 운영체제를 모두 지원한다. 하지만 다른언어에 비해서는 크게 뛰어난 장점은 없는 거 같고, 쓰는 사람도 많이는 없는거 같다. 그리고 설치가 무 료이긴 하지만 다른 언어들에 비해서 설치가 어렵다고 한다.

Javascript 가장 많이 들어보기도 했고, 가장 많이 궁금했던 언어이다.왜냐하면 지금 자바를 배우는데 처음에 자바랑 자바스크 립트의 차이를 몰라서 책을 살 때 잘 못 살뻔했다.자바 스크립트는 자바를 응용해서 만들었는데 1.자바에 비해 사 용하기에 쉽게 만들었고,2. 표준 HTML 문서를 만들때 사용하여 인터랙티브한 웹페이지를 만들 수 있게한다. 일단 자바스크립트는 2. 특정 환경에 구애 받지 않는다고 한다.그리고 3. 다른 언어들에 비해서 컴파일의 과정 없 이 바로 접근이 가능하다고한다. 4. window,유닉스,리눅스에 관계 없이 작동이 가능하고, 5. 명령어가 많지 않으 며, 문법도 c에 비해서 쉬운편이라고한다.
